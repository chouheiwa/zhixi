## README 结构

### 1. 标题 + 简介 + 徽章

- 项目名：知析
- 一句话描述：帮助知乎创作者深度分析致知计划收益数据，发现收益规律，预测未来趋势
- 徽章：Chrome 扩展、React、TypeScript、License

### 2. 功能亮点（6 个模块）

每个模块 1-2 句话 + 截图占位符，按使用路径排列：

1. **智能数据同步** — 自动采集收益、内容指标和实时汇总数据，支持多日批量同步
2. **多维度数据面板** — 20+ 可定制分析面板，支持拖拽排列
3. **ML 收益预测** — 随机森林 + 岭回归 + TensorFlow.js 神经网络集成学习
4. **内容深度分析** — 文章/回答/想法三类内容，多维排名，未变现内容识别
5. **RPM 预测与异常检测** — 千次阅读收益预测，异常波动发现
6. **收益目标追踪 & 数据导出** — 月度目标进度可视化，Excel 一键导出

### 3. 快速开始

**安装**（4 步）：
1. 克隆仓库
2. `npm install`
3. `npm run build`
4. Chrome 加载 dist/ 目录

**使用**（4 步）：
1. 登录知乎，点击工具栏图标
2. 点击「同步数据」
3. 点击「打开面板」进入仪表盘
4. 首次使用有引导教程

### 4. 技术架构 + 开发指南

**技术栈速览**：React 18 + TypeScript + Ant Design + Vite + ECharts + TensorFlow.js + Dexie

**三层架构**：
- Service Worker：消息路由、数据同步调度
- Content Script：fetch bridge 绕过 CORS
- UI 层：Popup + Dashboard

**开发命令**：dev / test / test:coverage / lint / format

**代码规范**：Husky + lint-staged 自动检查

### 5. 贡献指南

4 步流程：Fork → 创建分支 → 提交代码 → 提交 PR

提交规范：Conventional Commits（feat/fix/refactor/docs/test/chore）
